Pelajaran 6: ikon atau tag link

silahkan perhatikan icon pada tab peramban yang aktif, yang sama dengan gambar dibawah. Gambar dibuat menggunakan tag link yang ditempatkan didalam tag head

Tampilan HTML

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width"> <!--Tag untuk membuat icon website--> <link rel="icon" type="image/png" href="images/myicon.png"> <!----------------------------------> <title>Materi06</title> </head> <body> </body> </html>
